#### **⚙ Install**

1. Clone Project

```
  git clone https://github.com/nguyenhoaikhang37/BookingMovieCy.git
```

2. Install Packages

```
  npm install (or yarn install)
```

3. Run project

```
  npm start (or yarn start)
```
